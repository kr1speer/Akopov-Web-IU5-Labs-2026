# Лабораторная работа 2: Калькулятор на JavaScript

## Цель работы
Знакомство с инструментами построения пользовательских интерфейсов web-сайтов: HTML, CSS, JavaScript. В ходе выполнения работы вам предстоит продолжить реализовывать простой калькулятор и затем выполнить задания по варианту.

## Задание

1. **Запрограммируйте операцию смены знака (+/-)**
2. **Запрограммируйте операцию вычисления процента (%)**
3. **Запрограммируйте операцию вычисления квадратного корня (√)**
4. **Запрограммируйте операцию возведения в квадрат (x²)**
5. **Запрограммируйте операцию вычисления факториала (x!)**
6. **Добавьте кнопку стирания введенной цифры назад (backspace)**
7. **Добавьте кнопку, которая за раз добавляет сразу три нуля (000)**
8. **Запрограммируйте накапливаемое сложение**
9. **Запрограммируйте накапливаемое вычитание**
10. **Сделайте смену цвета фона по кнопке**
11. **Сделайте смену цвета окна вывода результата по кнопке**
12. **Добавьте в калькулятор вашу индивидуальную операцию**
